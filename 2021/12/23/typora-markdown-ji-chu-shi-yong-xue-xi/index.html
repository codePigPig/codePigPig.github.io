Typora+Markdown基础使用学习

Typora简介：

Typora给您既是读者又是作家的无缝体验。它删除了预览窗口，模式切换器，降价源代码的语法符号以及所有其他不必要的干扰。相反，它提供了真正的实时预览功能，可帮助您专注于内容本身。

Markdown简介：

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

一、标题：

简述：标题主要以#号表示，一个#号一级标题，##号为二级标题

# 一级标题        （快捷键：Command+1）

## 二级标题        （快捷键：Command+2）

### 三级标题        （快捷键：Command+3）

#### 四级标题    （快捷键：Command+4）

效果演示：

一级标题

二级标题

三级标题

四级标题

二、字体

简述：主要是用*、～～号将字体内容包起来，下面``是代码修饰， 实际修饰字体时不加

加粗：`**Hello,world**`        （快捷键：Command+B）

倾斜：`*Hello,world*`（快捷键：Command+I）

加粗+倾斜：`***Hello,world***`

删除线：`~~Hello,world~~`（快捷键：Control+Caps Lock+`）

下划线：`<u>Hello,world</u>`（快捷键：Command+U）

效果展示：

加粗：Hello,World!

倾斜：Hello,world!

加粗+倾斜：Hello,world

删除线：Hello,world

下划线：Hello,world

三、引用

简述：主要用到>符号，在需要引用的文字加上>

语法：
>引用内容

效果展示：

引用内容

四、分割线

简述：主要用到*和-符号

***
---

效果展示：

***：

—：

五、超链接

简述：主要用到[]、()

语法：
[超链接名字](超链接URL)
示例：
[我的个人网站](http://www.mywzb.com/)

效果展示：

我的个人网站

六、图片

简述：主要用到 !、[]、()

语法
![图片alt](本级图片地址/图片超链接)         （快捷键：control+command+U）
示例：

七、列表

无序列表：

简述：主要用到-、+、*

语法：
- 列表内容                        （快捷键：Option+Command+O）
+ 列表内容
* 列表内容

效果展示：

  • 列表内容
  • 列表内容
  • 列表内容
有序列表：

简述：主要用到数+符号.

语法：
1. 列表内容                        （快捷键：Option+Command+U）
2. 列表内容
3. 列表内容
注明：
在1.的.符号后空格，再按ENTER键实现有序1. 2. 3.

效果展示：

  1. 列表内容
  2. 列表内容
  3. 列表内容

八、表格

简述：Markdown使用代码方式生成表格过于麻烦，这里推荐使用快捷键选择行列生成表格

快捷键：Option+Command+T
鼠标选择：菜单栏-段落-表格-插入表格

效果展示：

九、代码

单行代码：
语法：
`代码内容`     （快捷键：Option+Command+C）

效果展示：

This is a single line code
代码块：
语法： 
​``` markdown                            
​``` java
​``` markdown
注明：
​```后面加上语言名称，可以选择代码块的代码类型（没有空格）

效果展示：

  system.out.println("Hello world");
