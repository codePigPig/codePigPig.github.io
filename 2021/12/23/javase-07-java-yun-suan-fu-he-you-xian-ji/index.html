Java运算符

Java语言支持如下运算符：

  • 算术运算符：+、-、*、/、%（取余）、++、- -

  • 赋值运算符：=

  • 关系运算符：>、<、<=、>=、==、!=instanceof

  • 逻辑运算符：&&（与）、||（或）、!（非）

  • 位运算符：&、｜、^、～、>>、>>>

  • 条件运算符：? : 三元运算

  • 扩展赋值运算符：+=、-=、*=、/=

注意事项：

  • 自动类型提升：

    • 当容量小的数据类型的变量与容量大的数据类型做运算时，结果自动提升为容量大的类型。
    • byte、char、short—>int—->long—>float—->double
    • 当byte、char、short三种类型用运算符运算后自动转型为int类型

  • 强制类型转换

    • 需要使用强转符（），括号中加入想转换的类型

代码演示：

  • 简单二元运算：
//二元运算
int a = 10;
int b = 20;

System.out.println(a+b);
System.out.println(a-b);
System.out.println(a*b);
System.out.println(a/(double)b);  //都是int类型相除后会有小数点，需要强制转换类型
System.out.println(a%2); //取余

输出：
30
-10
200
0.5
0
  • 关系运算：
int a = 10;
int b = 20;

System.out.println(a>b);
System.out.println(a<b);
System.out.println(a==b);
System.out.println(a!=b);

输出：
false
true
false
true

  • 自增自减（一元运算符）

    • a++、a–：先进行表达式运算，再进行自增或者自减运算，
    • ++a、–a：先进行自增或者自减运算，再进行表达式运算
    int a = 3;
int b = a++; //执行完这段代码后，先给b赋值b=a(3),再将a自增
//a = a+1;

System.out.println("a++自增后的a为："+a);
System.out.println("b为："+b);

int c = ++a; //执行这段代码后，先将a自增，再把给c赋值c=a(5)

System.out.println("++a自增后的a为："+a);
System.out.println("c为："+c);

输出：
a++自增后的a为：4
b为：3
++a自增后的a为：5
c为：5

  • 逻辑运算符

    • &&：逻辑与运算，两个变量都为真，结果才为true
    • ||：逻辑或运算，两个变量有一个为真，则结果才会true
    • ！：取反，如果是真，则变为假，如果是假，则变为真
    boolean a = true;
boolean b = false;

System.out.println("a&&b："+(a&&b));    //与
System.out.println("a||b："+(a||b));    //或
System.out.println("!(a&&b)："+!(a&&b)); //取反

输出：
a&&b：false
a||b：true
!(a&&b)：true

  • 位运算

    • &：与，全1为1，有0则0
    • ｜：或，有1则1，全0为0
    • ^：异或，不同为1，相同为0
    • ～：非，取反
    /*
   A = 0011 1100
   B = 0000 1101

   与，全1为1，有0则0
   A&B = 0000 1100

   或，有1则1，全0为0
   A|B = 0011 1101

   异或，不同为1，相同为0
   A^B = 0011 0001

   ～：非，取反
   ~B = 1111 010
 */

    • <<：左移运算

      • 数值value向左移动num位，左边二进制位丢弃，右边补0。（注意byte和short类型移位运算时会变成int型，结果要强制转换）
      • 若1被移位到最左侧，则变成负数。
      • 左移时舍弃位不包含1，则左移一次，相当于乘2

    • (>>)：右移运算，这里加括号是防止Tycora将>转换引用方式

      • 数值value向右移动num位，正数左补0，负数左补1，右边舍弃。（即保留符号位）
      • 右移一次，相当于除以2，并舍弃余数。
      • 无符号右移>>>：左边位用0补充，右边丢弃。
      /*
    0000 0000   0
    0000 0001   1
    0000 0010   2
    0000 0011   3
    0000 0100   4
    0000 1000   8
    0001 0000   16
 */

System.out.println("左移运算："+(2<<3));
System.out.println("右移运算："+(8>>2));

输出：
左移运算：16
右移运算：2

  • 三元运算：

    • 条件表达式?表达式1:表达式2
    • 其逻辑为：“如果条件表达式成立或者满足则执行表达式1，否则执行第二个。”常用在设置默认值,例如某个值不一定存在,则判断这个值是否存在,不存在给默认值(表达式2)。
    //x ? y : z
int score = 50;
String result = score < 60 ? "不及格" : "及格";
System.out.println(result);

输出：
不及格

判断是否属于同类型		 从左向右

细的解释		 从左向右

11 && 短路与 从左向右
