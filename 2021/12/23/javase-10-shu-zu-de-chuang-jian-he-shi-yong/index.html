数组

数组的概述：

  • 数组是相同类型数据的有序集合
  • 数组描述的是相同类型的若干个数组，按照一定的先后次序排列组合而成
  • 其中，每一个数据称做一个数组元素，每个数组元素可以通过一个下标来访问它们

java数组的特点：

  • 数组所有元素初始化为默认值，整型都是0，浮点型是0.0，布尔型是false
  • 数组一旦创建后，大小就不可改变
  • 数组中的元素可以是任何数据类型，包括基本类型和引用类型，但是数组中必须是相同类型，不允许出现混合类型
  • 数组变量属于引用类型，数组也可以看成是对象，数组中的每个元素相当于该对象的成员变量
  • Java中对象是在堆中的，因此数组无论保存原始类型还是其他对象类型，数组对象本身是在堆中的

数组的声明和创建

  • Java语言使用new操作符来创建数组，语法如下：
dataType[] arrayRefVar = new dataType[arraySize]
  • 数组的元素是通过索引访问的，数组索引从0开始
  • 获取数组长度：
    • arrays.length

代码演示：

//定义一个数组变量nums，
int[] nums = new int[5];
//数组对应下标设置赋值
nums[0] = 1;
nums[1] = 2;
nums[2] = 3;
nums[3] = 4;
nums[4] = 5;
//使用for循环输出数组中的值
for (int i = 0; i < nums.length; i++) {
    System.out.println(nums[i]);
}

输出：
1

数组的边界

  • 下标的合法区间：[0,length-1]，如果越界就会报错
  • ArrayIndexOutOfBoundsException: 数组下标越界异常

小结：

  • 数组是相同数据类型（可以为任意类型）的有序集合
  • 数组也是对象。数组元素相当于对象的成员变量
  • 数组长度是确定的，不可变的

数组的使用

遍历数组：

  • 查处最大值
//查找出最大值
int[] nums = {10,5,30,6,20,11};
int max = nums[0];
for (int i = 1; i < nums.length; i++) {
    if (nums[i]>max){
        max = nums[i];
    }
}
System.out.println("最大值为："+max);

输出：
30

For-each循环遍历：

  • 循环输出
int[] nums = {10,20,30,40,50};
for (int num : nums) {
    System.out.println(num);
}

输出：
10
20
30
40
50

数组作为方法入参和数组作返回值：

  • 打印和反转数组
public class Demo03 {
    
    //数组作为参数
    public static void printArray(int[] arrays){
        for (int array : arrays) {
            System.out.print(array+" ");
        }
    }

    //数组作为返回值
    public static int[] reverse(int[] arrays){
        int[] reverse = new int[arrays.length];

        for (int i = 0,j = reverse.length-1; i < arrays.length; i++,j--) {
            reverse[j]=arrays[i];
        }

        return reverse;
    }

    public static void main(String[] args) {

        //定义数组nums
        int[] nums = {1,2,3,4,5,6};
        //反转数组nums
        int[] reverse = reverse(nums);
        //打印数组reverse
        printArray(reverse);

    }
}

输出：
6 5 4 3 2 1

冒泡排序

概述：

  • 冒泡排序无疑是最为出名的排序算法之一
  • 冒泡的代码是两层循环，外层冒泡轮数，里层依次比较，时间复杂度O(n2)

代码演示：

public class Demo04 {
    //冒泡排序
    /*
        1. 比较数组中，两个相邻的元素，如果第一个数比第二个大，我们就交换它们的位置
        2. 每一次比较，都会产生出一个最大，或者最小的数字
        3. 下一轮则可以少一次排序
        4. 依次循环，直到结束
     */
    public static void main(String[] args) {

        int[] nums = {10,20,11,5,4,6};
        int[] sort = sort(nums);
        System.out.println(Arrays.toString(sort));


    }

    public static int[] sort(int[] array){

        int temp = 0;

        for (int i = 0; i < array.length-1; i++) {
            for (int j = 0; j < array.length-1-i; j++) {
                if (array[j]>array[j+1]){
                    temp = array[j];
                    array[j] = array[j+1];
                    array[j+1] = temp;
                }
            }
        }
        return array;
    }
}

输出：
[4, 5, 6, 10, 11, 20]

稀疏数组

概述：

  • 当一个数组中大部分元素为0，或者为同一值的数组时，可以用稀疏数组来保存该数组

稀疏数组处理方式：

  • 记录数组一共有几行几列，有多少个不同与0的值
  • 把具有不同值的元素和行列及值记录在一个小规模的数组中，从而缩小程序的规模
package array;

/**
 * @author CodePigPig
 * @date 2021/2/1 10:41 上午
 * @Email 1399203705@qq.com
 */
public class Demo05 {
    public static void main(String[] args) {

        //稀疏数组
        int[][] array1 = new int[11][11];
        array1[1][2] = 1;
        array1[2][3] = 2;
        System.out.println("===========原生数组===========");
        for (int[] ints : array1) {
            for (int anInt : ints) {
                System.out.print(anInt+" ");
            }
            System.out.println();
        }

        //遍历数组，查询出一共有多数个非0数据
        int count = 0;
        for (int i = 0; i < array1.length; i++) {
            for (int j = 0; j < array1[i].length; j++) {
                if (array1[i][j]!=0){
                    count++;
                }
            }
        }

        //创建并设置稀疏数组的首行数据
        int[][] array2 = new int[count+1][3];
        array2[0][0] = array1[0].length;
        array2[0][1] = array1.length;
        array2[0][2] = count;

        //遍历二维数组，将非零的值，存放在稀疏数组中
        int count2 = 1;
        for (int i = 0; i < array1.length; i++) {
            for (int j = 0; j < array1[i].length; j++) {
                if (array1[i][j]!=0){
                    array2[count2][0] = i;
                    array2[count2][1] = j;
                    array2[count2][2] = array1[i][j];
                    count2++;
                }
            }
        }

        System.out.println("===========稀疏数组============");
          for (int[] ints : array2) {
              for (int anInt : ints) {
                  System.out.print(anInt+" ");
              }
              System.out.println();
          }        


        System.out.println("===========还原数组============");

        //读取稀疏数组的首行，获取到行列
        int[][] array3 = new int[array2[0][0]][array2[0][1]];
        //遍历稀疏数组，还原值
        for (int i = 1; i < array2.length; i++) {
            array3[array2[i][0]][array2[i][1]] = array2[i][2];
        }
        for (int[] ints : array3) {
             for (int anInt : ints) {
                 System.out.print(anInt+" ");
             }
             System.out.println();
        }                                    

    }
}
